## ✨ What Makes This Portfolio Unique

This isn't just another portfolio website. It's an **interactive terminal experience** that lets you explore my work the way I work — through commands.

### 🎯 Key Features

1. **🖥️ Interactive Terminal**  
   - Real Unix-like shell interface with custom commands  
   - Type `help` to explore available commands  
   - Type `neofetch` to see my developer stats  
   - Auto-suggest and command history support  

2. **🎨 Premium Design System**  
   - Refined single dark theme (Ben 10 inspired)
   - Glassmorphism effects and neon shadows  
   - Fully responsive across all devices  
   - Minimal, cyberpunk-inspired aesthetic with premium micro-animations

3. **📂 Smart Project Navigation**  
   - Interactive project filtering (Web, Backend, Open Source)  
   - Click project names in the terminal to scroll and highlight cards  
   - Live demos and GitHub links for all projects  

4. **📊 Real-Time GitHub Stats**  
   - Live contribution graph powered by GitHub API  
   - Year-by-year contribution tracking  
   - Dynamic visual progress bars  

5. **⚡ Performance-Optimized**  
   - Built with Next.js 15 App Router  
   - Server-side rendering for SEO  
   - Optimized images and lazy loading

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/ts-31/tejas-sonawane-portfolio.git

# Navigate to project directory
cd tejas-sonawane-portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```bash
# Create optimized production build
npm run build

# Start production server
npm start
```
